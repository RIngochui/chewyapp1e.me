# Personal Website: ChewyOS

A fake retro Windows 98-style OS portfolio site built with React, Vite, and TypeScript.

**Live:** [ringochui.github.io/chewyapp1e-me](https://ringochui.github.io/chewyapp1e-me/)

## Features

- Draggable windows with Win98 chrome
- Desktop icons with rubber-band multi-select and drag-to-trash
- Start menu with app search
- MSN Messenger contact list
- Apple Browser with YouTube embed support
- Notepad, Resume viewer, Projects, About Me
- Recycle Bin with restore
- Dark mode toggle
- Mobile touch support
- CRT scanline overlay

## Stack

- React + TypeScript + Vite
- react-draggable
- CSS (custom Win98 theme)

## Run locally

```bash
npm install
npm run dev
```
