# chewyapp1e.me — CHEWYOS

Fake retro OS portfolio site. React + Vite + TypeScript + react-draggable + Tailwind CSS.

## iframe compatibility
Any app or game built for chewyapp1e.games or chewyapp1e.apps should NOT set `X-Frame-Options` or `Content-Security-Policy: frame-ancestors` headers, so it can be loaded inside 龍Browser (the fake in-OS browser). Do not add these headers in Vercel, Netlify, or any other deployment config for those projects.
